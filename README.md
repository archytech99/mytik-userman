# MyTIK User Manager Hotspot

## About MyTIK

- Simple web application for User Management Hotspot.
- Using framework Laravel 7.30.4

## Usage

- Copy file `.env.example` to `.env`
- Run command `php artisan key:generate` to create new `APP_KEY`
